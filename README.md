## 📌 Overview

### 🔹 Stage 2 (Introduction to Artificial Intelligence)
- Fundamentals of machine learning using Scikit-learn  
- Data preprocessing and exploratory data analysis (EDA)  
- Regression models and performance evaluation  
- Binary and multi-class classification techniques  
- Feature engineering and data preparation workflows  
- Introduction to PyTorch for tabular data  
- Building end-to-end machine learning pipelines (Breast Cancer case study)   

### 🔹 Stage 3 (Advanced AI Course)
- PyTorch fundamentals and tensor operations  
- Convolutional Neural Networks (CNNs) and AlexNet for image classification  
- Autoencoders for image generation (MNIST)  
- Data augmentation techniques for improving model generalization  
- Transfer learning and fine-tuning (Cats vs Dogs)  
- Image similarity search using EfficientNetV2  
- Image segmentation using U-Net (Custom & SMP implementations)  
- Object detection using Faster R-CNN and YOLO
